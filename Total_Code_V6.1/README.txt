config.h contains the following
	pin assignments (Motor Pins, Button Pins, etc.)
	constant values (Gear Ratios, Angle Tolerances, etc.)
	global variable declaration
	includes for libraries
	function prototype

config.cpp contains the following
	actual definitions for variables and their values
	this file is what takes the config.h and sets it be usable. 

DO NOT change these files without being sure how to do so.
